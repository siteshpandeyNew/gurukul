Documents Required

Please ensure you have the following documents ready before applying:

For New Admissions (Nursery / LKG)

  • Birth Certificate (original + photocopy)
  • Aadhar Card of the child
  • 4 passport size photographs of the child
  • Parent Aadhar Card (both parents)
  • Address proof (Aadhar/Votfer ID/Utility Bill)
  • Blood group certificate

For Transfer Admissions (Class II onwards)

  • All above documents
  • Transfer Certificate (TC) from previous school
  • Report card of the last class attended
  • Character certificate from previous school
  • Migration certificate (for Class XI from other board)
